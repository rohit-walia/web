Draw a pyramid


Rules:

Use only three characters: star *, forward-slash /, and backward-slash \

Should be perfectly symetrical

The tip of the pyramid should only consist of slashes and the base should only consist of stars. Everything in-between should have a combination of all three characters

Design the structure to be dynamic in size

You can't use nested for-loops

                private static final char BACKSLASH ='\\';
                private static final char STAR = '*';
                private static int counter = 0;

                public static void main(String[] args)

                  printPyramid(12);

                public static void printPyramid(int height)
                {
                  counter++;
                  int numberOfStarsToPrint = height*2-2;

                      if(i == Math.decrementExact(width/2))
                      {
                        write.print(SLASH);
                      }
                      else if(i == width/2)
                      {
                        write.print(BACKSLASH);
                      }
                      else
                        write.print(" ");
                    }

                  for (int i=0; i < width; i++)

                    }
